# BrickMoose Project Context

## Design Aesthetic (The "Anti-Slop" Manifesto)
* Style: Dark mode, minimalist, high-end, tech-industrial.
* Avoid: Generic "tech-startup" templates, over-symmetry, corporate marketing-speak, "childish" or "bouncy" animations.
* Preference: Intentional, heavy-feeling, precise animations (3D tilts, cursor spotlights, scroll-triggered reveals).
* Logic: Precision and engineering-focused copy. 

## Project Architecture
* Setup: Modular (separate index.html, style.css, main.js).
* Routing: Root domain (brickmoose.com) is custom HTML/CSS; Store (store.brickmoose.com) is Squarespace.
* Priority: Ease of maintenance; keep styles in CSS and logic in JS.

## Current Status
* Site: Hero section implemented with background image support and spotlight cards.
* Tech stack: Pure HTML/CSS/JS (no heavy frameworks).
